一个将 WebGAL 游戏打包到安卓平台的简易 DEMO。

本 DEMO 使用的游戏及图标均来自：[MakinoharaShoko/WebGAL](https://github.com/MakinoharaShoko/WebGAL)

## 如何打包你的游戏
* 安装 android studio 并导入本项目
* 将 `app\src\main\assets\webgal` 下的游戏换成你想要打包的游戏
* 更改包名以及游戏名和图标
* 点击菜单栏 `Build` -> `Generate Signed Bundle or APK` 构建apk
